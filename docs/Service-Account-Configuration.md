# Service Account Configuration

## Purpose

Service accounts provide identities for applications, automation workflows, and cloud services.

This project demonstrates secure service account management within Google Cloud Platform.

## Service Account Details

Name:

cloud-security-viewer

Description:

Service account used for IAM security and least-privilege access demonstration.

## Assigned Permissions

Viewer Role

Purpose:

- View project resources
- Inspect cloud configurations
- Verify access permissions

## Security Controls Implemented

### Least Privilege

Only required permissions were granted.

### Keyless Authentication

No service account keys were created.

### Centralized Access Management

Access was managed using IAM policies.

## Security Benefits

- Reduced credential exposure
- Simplified identity management
- Improved security posture
- Easier auditing and monitoring
